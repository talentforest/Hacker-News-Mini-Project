# 📰 Hacker News Mini Project

## 해커 뉴스 미니 프로젝트 소개

> 패스트캠퍼스 프론트엔드 수업에서 타 UX/UI디자인 수강생 분들과 협업하여 해커 뉴스 사이트를 만들어보는 프로젝트를 진행했습니다.

### 🔎 Specs

- React
- React-router-dom
- Hacker News API
- Axios

👀 React로 처음 프로젝트를 해보는만큼 더 잘 공부해야봐야겠다는 생각으로 상태 관리 라이브러리를 쓰지는 않았습니다.
